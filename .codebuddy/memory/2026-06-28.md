# 2026-06-28

## JMCL-for-servers 项目开发完成

完成了 Minecraft Java 版服务器启动管理器的全栈开发，包含三个核心模块：

### 1. 后端核心容器 (backend-core/)
- Java 21 + Spring Boot 3.2，16 个源文件
- McServerService: MC 服务器进程管理（启动/停止/重启/命令/Aikar's JVM flags）
- ServerInstanceManager: 服务器实例 CRUD + JSON 持久化
- DownloadService: 9 种服务端类型异步下载（VANILLA/PAPER/PURPUR/FOLIA/FABRIC/QUILT/FORGE/NEOFORGE/VELOCITY）
- ConfigService: server.properties 读写，完整的 eula/ops/whitelist/banned 配置管理
- ServerTypeRegistry: 9 种服务端类型的 API 配置注册
- 3 个 REST Controller: ServerController/DownloadController/ConfigController

### 2. 前端容器 (frontend/)
- TypeScript + React 18 + MD3 设计规范
- 完整的 MD3 主题系统（lightTheme/darkTheme，160+ 设计令牌）
- 8 个组件: Button(4 变种)/Card/TextField/Dialog/Snackbar/Spinner/Sidebar/Terminal
- 5 个页面: DashboardPage/ServerListPage/ServerConfigPage/DownloadPage/LogsPage
- Zustand UI 状态管理（主题切换/Snackbar/Sidebar 折叠）
- nginx 反向代理配置，前端 27781 端口，后端 API 代理

### 3. Java 安装器 (installer/)
- 跨平台自动安装（Linux systemd / macOS launchd）
- OsDetector: 操作系统/包管理器/WSL 检测
- DockerInstaller: 自动安装 Docker（Linux get.docker.com / macOS brew cask）
- ServiceConfigurator: 配置系统服务实现开机自启
- 一键安装流程：检测→安装Docker→复制文件→配置服务→启动容器

### 容器编排
- docker-compose.yml: backend-core + frontend 两个服务
- Docker volume 持久化数据
- Docker socket 挂载用于 MC 服务器进程管理
